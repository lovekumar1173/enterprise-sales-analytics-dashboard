# Enterprise Sales Analytics & Risk Dashboard

This project is an enterprise-level analytics and risk assessment dashboard
designed to simulate a real consulting engagement.

It analyzes sales performance, identifies high-risk customers, and highlights
payment delays using structured data and visual dashboards.

## Tech Stack
- Python
- MySQL
- SQL
- Matplotlib
- XAMPP

## Key Features
- Normalized multi-table database design
- Advanced SQL analytics using joins and aggregations
- Financial risk and payment delay analysis
- Python-based visual dashboard for business insights
- Consulting-style documentation (BRD & TDD)

## Dashboard Insights
- Revenue performance by product
- Identification of high-risk customers
- Payment delay analysis for financial risk assessment
